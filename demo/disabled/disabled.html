Disabling

There is an attribute "disabled" on ui-scroll which allows to ignore user scroll/resize events: 
<div ui-scroll="item in datasource" disabled="needToDisable">{{item}}</div>
Here we have two-way binding, so just change appropriate property on appropriate scope: 
$scope.needToDisable = true;
Since "disabled" is true no user scroll is being processed by ui-scroll engine.
User scroll processing is {{needToDisable ? "disabled" : "enabled"}}
